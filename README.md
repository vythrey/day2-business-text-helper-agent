# Business Text Helper Agent

## Project overview
This is a beginner Python agent that helps process business text using simple rule-based logic.

The agent can:
- summarize text
- rewrite text in a more professional form
- convert text into bullet points
- remember the last task used

## Why I built this
I wanted to move from a very basic assistant agent to a more practical business-use case that reflects real workplace needs like summarization, documentation, and communication support.

## Features
- Intent-based command detection
- Text summarization
- Professional rewriting
- Bullet point extraction
- Basic memory using Python dictionary
